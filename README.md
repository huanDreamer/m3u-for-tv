# 电视直播 IPTV

## 支持安卓手机、安卓平板、iOS、安卓TV 等设备，理论上也支持 ipad， appleTv 没有做适配

## 使用方式

### 触屏设备 手机 / pad 等

![phone-1.png](source%2Fphone-1.png)


1. 点击屏幕中间调出左侧频道选择菜单
2. 点击频道名字换台

### 非触屏设备（电视）
![tv-1.png](source%2Ftv-1.png)

1. 按 确认 / 上 / 下 键 调出左侧频道菜单
2. 使用 上 / 下 键进行频道选择
3. 再次按 确认 键换台
4. 连续两次按返回键退出app
